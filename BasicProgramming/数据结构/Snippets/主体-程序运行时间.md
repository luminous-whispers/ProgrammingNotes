- 名称: clock_main_body_windows_only
- 作用: 统计当前程序运行时间
- 注意:
1. 仅在windows下可以使用

```c
 #include<time.h>
 int main()
 {       
	 clock_t start, end; 
      start = clock(); 
      /*...
         需要计时的代码
     ...*/
	 end = clock();
     printf("time=%f\n", (double)(end - start) / CLOCKS_PER_SEC);
	return 0;
}
```
